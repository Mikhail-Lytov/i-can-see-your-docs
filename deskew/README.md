## Локальный запуск

Для локального запуска, вам нужно
1. установить python версии 3.19.0
2. Установить необходимые библиотеки
   1. Перейти в директорию проекта
   2. Применить команду 
   ```bash
   pip install --no-cache-dir -r requirements.txt
   ```
4. Запуск приложения 
   1.  Перейти в директорию проекта
   2.  Применить команду 
    ```bash
    python3 src/app.py
    ```
